# TYBA - Asistente de IA para Sistema Justicia XXI Web

## Introducción

TYBA es un asistente de IA especializado en el Sistema de Gestión de Procesos Judiciales **Justicia XXI Web (Tyba)**, desarrollado para brindar apoyo y orientación sobre el uso de esta plataforma judicial electrónica implementada gradualmente en todos los despachos y juzgados judiciales de Colombia.

### Funcionalidades del Asistente

1. **Consultas sobre Documentos Electrónicos Judiciales**
   - Explica qué son los documentos electrónicos judiciales (e-documents)
   - Orientación sobre creación y manejo de contenido electrónico
   - Ejemplo: "¿Cómo se crean los documentos electrónicos en el sistema?"

2. **Información sobre Actuaciones Judiciales**
   - Describe el índice del expediente judicial y sus componentes
   - Explica el orden cronológico y estampado de tiempo requerido
   - Ejemplo: "¿Qué elementos debe contener una actuación judicial en TYBA?"

3. **Orientación sobre Memoriales Electrónicos**
   - Define qué documentos se consideran memoriales
   - Explica el proceso de presentación posterior a la demanda
   - Ejemplo: "¿Cómo presento un recurso a través del sistema electrónico?"

4. **Gestión de Notificaciones Electrónicas**
   - Informa sobre los medios electrónicos autorizados
   - Explica los requisitos legales mínimos para notificaciones
   - Ejemplo: "¿Qué medios puedo usar para notificaciones electrónicas?"

5. **Información sobre Firmas Electrónicas**
   - Explica el sistema de firma del expediente
   - Orienta sobre responsabilidades y manejo de firmas electrónicas
   - Ejemplo: "¿Cómo funciona la firma electrónica en los expedientes?"

6. **Consultas sobre Términos y Plazos**
   - Aclara las reglas de recepción fuera de horario hábil
   - Explica excepciones como Habeas Corpus (36 horas)
   - Ejemplo: "¿Qué pasa si presento un memorial después de horas hábiles?"

7. **Soporte para Correos Institucionales**
   - Orienta sobre el uso de correos electrónicos autorizados
   - Explica la transmisión de comunicaciones electrónicas
   - Ejemplo: "¿Cómo configuro las notificaciones por correo institucional?"

## ⚠️ Limitaciones

- TYBA proporciona información general sobre el sistema y no constituye soporte técnico especializado
- La información se basa en el manual de procedimientos vigente hasta agosto de 2023
- Para problemas técnicos complejos o fallas del sistema, siempre es recomendable contactar al soporte técnico oficial
- El análisis de procedimientos es informativo y no reemplaza la capacitación oficial del sistema
- No puede resolver problemas de conectividad, acceso o configuración avanzada del sistema

## 💼 Mensaje Final

TYBA busca facilitar la comprensión y uso del Sistema Justicia XXI Web, apoyando la transformación digital del sistema judicial colombiano hacia el modelo de "cero papeles". Este asistente proporciona información valiosa basada en la documentación oficial, pero recuerda que cada situación puede requerir consulta con el soporte técnico oficial del sistema.

La implementación del Sistema Justicia XXI Web representa un avance significativo hacia mayor acceso al sistema de justicia, mayor eficiencia en los procesos y mejor gestión de documentos judiciales en ambiente web.

*"La transformación digital del sistema judicial colombiano busca brindar mayor acceso a la justicia y optimizar la gestión de procesos judiciales a través de herramientas tecnológicas modernas y eficientes."*
